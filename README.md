# Customer Support Operations Dashboard

## Project Purpose

This project aims to provide a centralized Power BI dashboard for monitoring customer support operations across products, departments, and agents.

## Project Structure

IT_Dashboard/

* raw_data/

  * Source files received from operational systems.

* processed_data/

  * Cleaned and transformed datasets used for reporting.

* Scripts/

  * Python scripts used for data preparation and dimension table creation.

* Documentation/

  * Project documentation and design artifacts.

* PowerBI/

  * Power BI report files and related assets.

## Core Datasets

* FactCalls
* FactAgentActivity
* FactAgentDaily
* FactDepartmentDaily
* DimProduct
* DimDepartment
* DimAgent
* DimDate
